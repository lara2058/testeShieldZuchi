# testeShieldZuchi

## Programa para teste dos shields zuchi

### Informações importantes

* Pinos de comando do Shield:
Os pinos são referentes a pinagem do Arduino Uno
	* DIR_MOTOR_A = D2;
	* PWM_MOTOR_A = D6;
	* DIR_MOTOR_B = D3;
	* PWM_MOTOR_B = D9.

* Conexão com o Arduino Nano:
	* DIR_MOTOR_A = D11;
	* PWM_MOTOR_A = D6;
	* DIR_MOTOR_B = D8;
	* PWM_MOTOR_B = D9.
